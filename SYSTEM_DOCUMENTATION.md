# 中国陶行知研究会儿童发展与教育专委会管理系统

## 系统概述

本系统是为中国陶行知研究会儿童发展与教育专委会开发的会员管理和课题管理平台，主要功能包括：

1. **会员管理**：会员注册、会员查询、会员收费管理
2. **课题管理**：课题申报、课题评估、课题立项、课题开展成果汇报
3. **用户认证**：登录、权限管理
4. **后台管理**：用户管理、会员管理、课题管理、系统设置

## 技术架构

### 前端技术栈
- React 18
- TypeScript 5
- Ant Design 5
- React Router 6
- Vite 5

### 系统架构
- **前端**：单页应用（SPA）
- **后端**：待开发（可使用 Node.js + Express + MongoDB 或其他技术栈）
- **部署**：静态文件部署到 Nginx 或其他静态文件服务器

## 功能模块

### 1. 会员管理

#### 1.1 会员注册
- 个人会员注册：填写姓名、性别、身份证号、联系电话、电子邮箱、工作单位、职务、通讯地址
- 单位会员注册：填写单位名称、单位级别、法定代表人、联系人、联系电话、电子邮箱、单位地址、单位简介
- 缴费方式：银行转账（转入中国陶行知研究会总会账户）
- 转账凭证上传

#### 1.2 会员查询
- 按姓名/单位名称查询
- 按会员类型查询
- 按联系电话查询
- 按状态查询

#### 1.3 会员收费管理
- 个人会员：60元/月，一次性缴纳3年，共计2160元
- 单位会员：
  - 一般单位会员：1000元/年
  - 委员（理事）单位：3000元/年
  - 副理事长单位：5000元/年
- 缴费凭证审核

### 2. 课题管理

#### 2.1 课题申报
- 填写课题名称、课题类型、申报人、联系电话、电子邮箱、工作单位、课题周期、课题摘要、研究内容、预期成果、课题经费
- 上传申报书

#### 2.2 课题评估
- 课题评分
- 评估状态管理

#### 2.3 课题管理
- 课题状态管理
- 进度跟踪
- 成果管理

### 3. 用户认证

#### 3.1 登录
- 用户名和密码登录

#### 3.2 权限管理
- 基于角色的权限控制

### 4. 后台管理

#### 4.1 用户管理
- 用户添加、编辑、删除
- 角色管理

#### 4.2 会员管理
- 会员信息管理
- 会员状态管理

#### 4.3 课题管理
- 课题信息管理
- 课题状态管理

#### 4.4 系统设置
- 系统基本配置

## 页面结构

### 主要页面
1. **登录页面**：用户登录入口
2. **首页**：系统概览
3. **会员注册**：会员信息填写和提交
4. **会员查询**：会员信息搜索和展示
5. **会员收费管理**：缴费记录管理和审核
6. **课题申报**：课题信息填写和提交
7. **课题评估**：课题评分和评估管理
8. **课题管理**：课题状态管理和进度跟踪
9. **后台管理**：系统管理功能

### 路由结构
- `/login`：登录页面
